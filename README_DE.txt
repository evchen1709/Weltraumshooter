
Tills Weltraum Shooter – Schnellstart

Variante A: Direkt lokal spielen
1) ZIP entpacken.
2) Doppelklick auf „Tills Weltraum Shooter.html“.
   WICHTIG: Nicht in der Vorschau von Mail/WhatsApp/Drive öffnen – dort wird JavaScript blockiert.
   Immer im richtigen Browser öffnen (Chrome/Firefox/Edge/Safari).

Variante B: Mit einem Link teilen (empfohlen)
1) Lade die Datei „index.html“ auf irgendeinen Webspace hoch (z. B. deine Homepage, GitHub Pages, Netlify, Vercel).
2) Teile die URL – dann läuft das Spiel sofort im Browser per Klick.

Warum klappt es in Mail/WhatsApp oft nicht?
– Diese Vorschaufenster laufen in einer Sicherheits-Sandbox und blockieren Skripte.
  Deshalb muss die Datei heruntergeladen und im Browser geöffnet oder online gehostet werden.
